# Mason L'Etoile
Software developer working across C++ engine systems, web tooling, development tools, and full-stack applications. Currently open to job offers, project ideas, and collaborations.

## My Projects
- [gh-top-languages](https://github.com/gh-top-languages): TypeScript GitHub language chart tooling for READMEs and websites: SVG rendering library, deployable API engine, and demo sandbox
- [star-setup](https://github.com/star-setup): A personal Rust CLI tool to clone, configure, and wire single or multi-repo ecosystems
- [contact-api](https://github.com/contact-api): Modular TypeScript contact form ecosystem: core logic, Resend and Nodemailer providers, and a Vercel deployment adapter
- [starweb-libs](https://github.com/starweb-libs): TypeScript 2D web engine: canvas, physics, tween, audio, and UI libraries powering personal browser games
- [starlet-libs](https://github.com/starlet-libs): Modular C++20/OpenGL engine ecosystem: rendering, ECS scenes, math, serialization, and tooling
